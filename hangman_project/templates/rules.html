How to Play Hangman

  • The game randomly selects a word from a chosen category and difficulty.
  • You must guess the word one letter at a time.
  • Each incorrect guess costs you a life.
  • You have 6 lives. If you lose all lives, the game is over.
  • If you guess all the letters before running out of lives, you win!
  • Repeated guesses of the same letter do not cost extra lives.
  • Use the reset button to start a new game with the same settings.
